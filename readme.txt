Social Feed Lite

== Description ==

Lightweight plugin to show Instagram posts.

== Frequently Asked Questions ==

= Do I need an API key to use this plugin =

No you just need to log into your instagram account and provide access.

= How can I display the posts =

You can use shortcode [social_feed_lite] to display instagram posts.

= Are there any customization allowed to the feed display =

Yes you can change the number of posts displayed.
